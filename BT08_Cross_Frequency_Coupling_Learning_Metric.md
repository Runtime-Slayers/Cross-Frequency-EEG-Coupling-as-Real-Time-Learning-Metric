## 2. Background: Brain Oscillations and Memory

### 2.1 What Are Brain Oscillations?

```
Neurons don't fire randomly. Large populations of neurons synchronize 
their activity, creating rhythmic electrical patterns = oscillations.

Each frequency band reflects a different brain process:

Delta (0.5-4 Hz): Deep sleep consolidation
   └── Imagine: ocean waves, very slow, very powerful

Theta (4-8 Hz): Memory encoding, exploration, emotional processing
   └── Imagine: walking rhythm, each step = one theta cycle
   └── Generated primarily by the HIPPOCAMPUS

Alpha (8-13 Hz): Inhibition, idling, attentional gating
   └── Imagine: "standby mode" for brain regions not in use

Beta (13-30 Hz): Active processing, motor control, status quo maintenance
   └── Imagine: busy working rhythm

Gamma (30-100 Hz): Feature binding, consciousness, detailed processing
   └── Imagine: rapid-fire coding, each burst carries specific information
   └── Generated by local cortical circuits
```

### 2.2 Cross-Frequency Coupling (CFC) — The Secret of Learning

```
KEY DISCOVERY (Lisman & Jensen, 2013 — one of the most cited 
neuroscience papers of the decade):

During learning, theta and gamma don't just coexist — they INTERACT:

THETA wave (one cycle ≈ 125-250ms at 4-8 Hz):
   ┌──────╱╲──────╱╲──────╱╲──────╱╲──────┐
   │     ╱  ╲    ╱  ╲    ╱  ╲    ╱  ╲     │
   │────╱────╲──╱────╲──╱────╲──╱────╲────│
   │   ╱      ╲╱      ╲╱      ╲╱      ╲   │
   └──╱────────╲───────╲───────╲────────╲──┘
       peak    trough   peak   trough

GAMMA bursts (30-100 Hz) are NESTED within each theta cycle:
   
   Theta peak:  [γ γ γ γ γ γ γ] ← 5-8 gamma cycles per theta peak
   Theta trough: [    quiet    ] ← gamma suppressed

This means: Each theta cycle can carry 5-8 "items" of information
encoded in gamma bursts. This is WHY working memory capacity is 7±2!

   7±2 items in working memory = 7±2 gamma bursts per theta cycle
   
This isn't coincidence — it's the MECHANISM.
```

### 2.3 Phase-Amplitude Coupling (PAC) — The Technical Name

```
"Phase-Amplitude Coupling" (PAC):
   The PHASE of a slow oscillation (theta)
   modulates the AMPLITUDE of a fast oscillation (gamma)

In math:

   Gamma_amplitude(t) = f(Theta_phase(t))

When PAC is strong:
   - Gamma bursts ONLY at specific theta phases (usually near peak)
   - = Well-organized neural processing
   - = Effective learning

When PAC is weak:
   - Gamma fires randomly, not locked to theta
   - = Disorganized processing
   - = Poor learning, poor memory formation

MEASUREMENT METHODS:
1. Modulation Index (MI) — Tort et al., 2010
2. Mean Vector Length (MVL) — Canolty et al., 2006
3. Phase-Locking Value (PLV) — Lachaux et al., 1999
4. Event-related coupling (not phase-dependent)
```

## 4. Computing the Modulation Index (MI)

### 4.1 Step-by-Step Mathematical Derivation

**Step 1: Extract theta phase and gamma amplitude**

```python
import numpy as np
from scipy.signal import butter, filtfilt, hilbert

# Raw EEG signal: x(t), sampled at fs = 256 Hz
fs = 256
x = eeg_data  # shape: (n_samples,)

# Bandpass filter for theta (4-8 Hz)
b_theta, a_theta = butter(4, [4, 8], btype='bandpass', fs=fs)
theta_filtered = filtfilt(b_theta, a_theta, x)

# Bandpass filter for gamma (30-80 Hz)
b_gamma, a_gamma = butter(4, [30, 80], btype='bandpass', fs=fs)
gamma_filtered = filtfilt(b_gamma, a_gamma, x)

# Hilbert transform to get instantaneous phase and amplitude
theta_analytic = hilbert(theta_filtered)
theta_phase = np.angle(theta_analytic)  # range: [-π, π]

gamma_analytic = hilbert(gamma_filtered)
gamma_amplitude = np.abs(gamma_analytic)  # envelope
```

**Step 2: Compute Modulation Index (Tort Method)**

```python
def modulation_index(theta_phase, gamma_amplitude, n_bins=18):
    """
    Tort et al. (2010) Modulation Index.
    
    Idea: If gamma amplitude depends on theta phase, then the distribution
    of mean gamma amplitude across phase bins will be NON-UNIFORM.
    
    MI = KL divergence between this distribution and uniform distribution
    
    Args:
        theta_phase: instantaneous phase of theta (radians)
        gamma_amplitude: instantaneous amplitude of gamma
        n_bins: number of phase bins (default 18 = 20° each)
    
    Returns:
        MI: Modulation Index (0 = no coupling, 0.5 = strong coupling)
    """
    # Bin phase into n_bins equal segments
    phase_bins = np.linspace(-np.pi, np.pi, n_bins + 1)
    
    # Compute mean gamma amplitude in each phase bin
    mean_amp = np.zeros(n_bins)
    for i in range(n_bins):
        mask = (theta_phase >= phase_bins[i]) & (theta_phase < phase_bins[i+1])
        if mask.sum() > 0:
            mean_amp[i] = gamma_amplitude[mask].mean()
    
    # Normalize to make it a probability distribution
    if mean_amp.sum() == 0:
        return 0.0
    p = mean_amp / mean_amp.sum()
    
    # Replace zeros with tiny value to avoid log(0)
    p[p == 0] = 1e-10
    p = p / p.sum()
    
    # Uniform distribution
    q = np.ones(n_bins) / n_bins
    
    # KL divergence
    KL = np.sum(p * np.log(p / q))
    
    # Modulation Index = KL normalized by max possible KL (= log(N))
    MI = KL / np.log(n_bins)
    
    return MI

# Compute
mi_value = modulation_index(theta_phase, gamma_amplitude)
print(f"Modulation Index: {mi_value:.4f}")

# Interpretation:
# MI < 0.001: No coupling (random)
# MI = 0.001-0.005: Weak coupling (passive listening)
# MI = 0.005-0.02: Moderate coupling (active engagement)
# MI = 0.02-0.05: Strong coupling (deep learning)
# MI > 0.05: Very strong coupling (insight moments)
```

### 4.2 Why MI Works

```
MATHEMATICAL INTUITION:

If gamma amplitude is CONSTANT across all theta phases:
   mean_amp = [5, 5, 5, 5, 5, 5, 5, 5, ...]
   p = [1/18, 1/18, ...]  = uniform
   KL divergence from uniform = 0
   MI = 0  → NO COUPLING
   
If gamma amplitude is HIGH only at theta peak (phase = 0):
   mean_amp = [0, 0, 0, 8, 10, 8, 0, 0, ...]
   p = [0, 0, 0, 0.31, 0.38, 0.31, 0, 0, ...]
   KL divergence from uniform = LARGE
   MI = LARGE → STRONG COUPLING
   
The more concentrated gamma is at specific theta phases,
the higher MI, the better the brain is organizing information.
```

## 5. The Complete Learning Quality Index (LQI)

### 5.1 Components

```
LQI = w₁ × MI_normalized + 
      w₂ × PAC_consistency +
      w₃ × GammaBurst_count_normalized +
      w₄ × Theta_power_normalized +
      w₅ × Alpha_suppression_normalized

Where:
─────────────────────────────────────────────────────────
Component 1: MI_normalized
   = Modulation Index / MI_max_personal_baseline
   Reflects: Strength of theta-gamma coupling
   High = organized memory encoding

Component 2: PAC_consistency
   = 1 - std(MI over sliding windows) / mean(MI over windows)
   Reflects: How STABLE is the coupling over time
   High = sustained deep processing (not flickering in and out)

Component 3: GammaBurst_count
   = Number of gamma bursts (>2× baseline) per theta cycle
   Range: 0-8 bursts per cycle
   Reflects: How many "items" are being processed per cycle
   High = rich information processing

Component 4: Theta_power_relative
   = Theta power / Total power (4-8 Hz / 1-40 Hz)
   Reflects: Is the hippocampal memory system engaged?
   High = active memory encoding

Component 5: Alpha_suppression (task vs rest)
   = (Alpha_rest - Alpha_task) / Alpha_rest
   Reflects: Is the cortex actively processing (not idling)?
   High = brain is "on" not "standby"

Weights (from literature correlations with retention):
   w₁ = 0.30 (MI is the primary driver)
   w₂ = 0.15 (consistency matters)
   w₃ = 0.20 (gamma bursts = information richness)
   w₄ = 0.20 (theta power = memory system engagement)
   w₅ = 0.15 (alpha suppression = cortical activation)
   Σ = 1.0
```

    p[p < 1e-10] = 1e-10
    p = p / p.sum()
    KL = np.sum(p * np.log(p * n_bins))
    return KL / np.log(n_bins)

MI = compute_MI(theta_phase, gamma_amplitude)
print(f"   MI = {MI:.4f}")
print(f"   Expected: >0.01 for coupled signal")
assert MI > 0.005, "MI too low for coupled signal!"
print("   [PASS]")

# Compute in 30-second windows
window = 30 * fs
n_windows = N // window
MI_windows = []
for w in range(n_windows):
    seg_phase = theta_phase[w*window:(w+1)*window]
    seg_amp = gamma_amplitude[w*window:(w+1)*window]
    MI_windows.append(compute_MI(seg_phase, seg_amp))

MI_mean = np.mean(MI_windows)
MI_consistency = 1 - np.std(MI_windows) / (MI_mean + 1e-10)

# Gamma bursts per theta cycle
theta_period = 1/theta_freq
samples_per_cycle = int(theta_period * fs)
gamma_threshold = np.mean(gamma_amplitude) + 2*np.std(gamma_amplitude)
burst_count = np.sum(gamma_amplitude > gamma_threshold) / (duration * theta_freq)

# Theta relative power
from scipy.signal import welch
freqs_psd, psd = welch(eeg, fs=fs, nperseg=512)
theta_mask = (freqs_psd >= 4) & (freqs_psd <= 8)
total_mask = (freqs_psd >= 1) & (freqs_psd <= 40)
theta_relative = psd[theta_mask].sum() / psd[total_mask].sum()

# Alpha suppression (simulate: during task alpha should be low)
alpha_mask = (freqs_psd >= 8) & (freqs_psd <= 13)
alpha_relative = psd[alpha_mask].sum() / psd[total_mask].sum()
alpha_suppression = max(0, 0.5 - alpha_relative) / 0.5  # assume rest alpha = 50%

# LQI
LQI = (0.30 * min(MI_mean/0.05, 1.0) + 
       0.15 * max(MI_consistency, 0) +
       0.20 * min(burst_count/8.0, 1.0) +
       0.20 * min(theta_relative/0.3, 1.0) +
       0.15 * alpha_suppression) * 100

print(f"   MI (mean): {MI_mean:.4f}")
print(f"   PAC consistency: {MI_consistency:.2f}")
